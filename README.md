# Topic Angle

把 AI / 科技素材，变成适合中文圈发布的选题角度。

`topic-angle` 是一个面向中文科技创作者的 Codex Skill。它不只是总结素材，而是帮助你判断：

- 这条内容值不值得发？
- 中文读者真正关心什么？
- 最有传播力、同时又站得住脚的切入角度是什么？
- 应该写成单帖、长线程、X Article，还是信息图？
- 哪些信息还需要事实核查？

> 它回答的不是“这条新闻讲了什么”，而是“为什么现在值得发，以及应该怎么发”。

## 它能做什么

给它一条 AI、科技、产品、商业、创业、模型发布、工具或开发者工作流素材，它会：

1. 判断素材是否值得发布；
2. 从信息差、实用性、争议性、新鲜度、传播性、中文圈适配度和配图潜力七个维度评分；
3. 提炼最适合中文圈的核心角度；
4. 找出普通用户、创作者、开发者和 AI 工具用户最关心的影响；
5. 生成五种不同类型的标题或开头；
6. 推荐单帖、主帖加自我回复、长线程、文章、信息图、多图卡片或视频口播稿；
7. 写出可直接继续打磨的主帖开场；
8. 给出后续回复方向、事实核查风险和配图方案。

## 适合谁

- 中文科技 X / Twitter 创作者
- AI、产品和开发者内容创作者
- 科技媒体、Newsletter 和社区运营者
- 需要把产品更新转化成用户视角内容的人
- 经常遇到“素材很多，但不知道该从哪里写”的人

## 它不会做什么

- 不把官方公告机械翻译一遍；
- 不为了流量把推测写成事实；
- 不默认使用“震撼发布”“彻底颠覆”“王炸”等廉价话术；
- 不只给一堆标题，而不解释为什么这个角度值得写。

## 安装

将仓库克隆到 Agent Skills 目录：

```bash
mkdir -p ~/.agents/skills
git clone https://github.com/JustWin33/topic-angle.git ~/.agents/skills/topic-angle
```

然后重启 Codex，让它发现新 Skill。

更新已安装的版本：

```bash
git -C ~/.agents/skills/topic-angle pull
```

## 使用方法

最简单的调用方式：

```text
使用 $topic-angle，帮我判断这个素材适不适合发 X，并提炼中文圈传播角度。
```

需要更明确的输出时：

```text
使用 $topic-angle，按中文科技 X 创作者视角分析这条素材：
1. 判断值不值得发
2. 提炼最适合中文圈的角度
3. 给 5 个主帖开头
4. 判断适合单帖、长线程还是信息图
5. 提醒哪些地方需要事实核查

[把素材粘贴在这里]
```

也可以只请求其中一部分：

```text
使用 $topic-angle，只给我最强传播角度、三个开头和风险提醒。
```

## 默认输出

完整分析包含九个部分：

1. 是否值得发
2. 最适合中文圈的角度
3. 普通人最关心什么
4. 可写标题 / 开头
5. 推荐发布形式
6. 推荐主帖切入
7. 后续自我回复方向
8. 风险提醒
9. 配图建议

## 设计原则

- 角度优先，而不是摘要优先；
- 读者影响优先，而不是官方措辞优先；
- 可信度优先，而不是夸张感优先；
- 明确区分已确认事实、合理推断和未经证实的信息；
- 最终角度必须能够用一句自然的中文讲清楚。
